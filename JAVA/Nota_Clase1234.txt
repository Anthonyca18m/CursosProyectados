Tipos de Datos:

Enteros.- 
	Int : 4 bytes de espacio para almacenamiento. Desde -2,147,483,648 hasta 2,147,483,648
	Short : 2 bytes de espacio para almacenamiento. Desde -32,768 hasta 32,767
	long : 8 bytes de espacio para almacenamiento. Desde ... Sufijo " l " Ejemplo: 864488795445l
	Byte : 1 byte de espacio para almacenamiento . Desde -128 hasta 127
	
Coma flotante ( decimales )

	Float : 4 bytes de espacio para almacenamiento. Aproximadamente 6 a 7 cifras decimales.Sufijo "F" ejemplo: 1.3546547f
	Double: 8 bytes de espacio para almacenamiento. Aproximadamente 15 cifras decimales
	
Char : para representar caracteres
 
Boolean	: solo tienen dos valoras  = true => verdadero , false => falso

String : para representar letras,números,etc.

¿Qué es una variable en Java?

espacio en la memoria del ordenador donde se almacenara un valor que podra cambiar durante la ejecucion del programa.

¿Porque lo utilizamos?
Porque a la hora de crear un programa surge la necesidad de guardar datos temporales que necesitaras en el futuro.

¿Como se crea una variable? 
Especificando el tipo de dato que almacenara en su interior + el nombre de la variable: Ejemplo: int salario;

¿Que es inicializar una variable?

Es darle valor. Nombre_variable_valor. P. ejemplo: salario = 2000;
Java no permite utilizar variables que no se hayan inicializado.
